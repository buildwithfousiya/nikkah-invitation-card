# Nikkah Invitation Card

A beautiful, premium, and fully responsive digital invitation website for the Nikkah ceremony of Haseeb & Naila.

## Features
- Fully responsive on mobile, tablet, and desktop viewports.
- Interactive Swipe-to-Attend entry screen.
- Ambient theme-matching background.
- Integrated background nasheed audio.
- Add-to-Calendar Google Calendar utility.
- Native sharing and direct wedding card download fallback.
- Localized styling and layout.
